# Prerequisites

## Required Software

| Tool | Version |
|---|---|
| Python | 3.10+ |
| Node.js | 18+ |
| npm | 9+ |
| Git | latest |
| Docker (optional) | latest |

## Recommended Environment

- OS: Windows, Linux, or macOS
- RAM: 8 GB+
- Free disk: 3 GB+

## Backend Dependencies

Install Python packages from:

```bash
pip install -r requirements.txt
```

## Frontend Dependencies

```bash
cd frontend
npm install
```
