# Release Notes

## Versioning

This project follows semantic versioning:
- MAJOR: breaking changes
- MINOR: backward-compatible features
- PATCH: backward-compatible fixes

---

## [1.0.0] - 2026-04-01

### Added
- Core HR modules (employees, leave requests, evaluations)
- Authentication with JWT
- Reporting module with exports
- Admission and termination workflows

### Improved
- Documentation structure and readability
- MkDocs rendering consistency

### Fixed
- Documentation links and routing behavior for published static pages
- Markdown rendering issues in multiple pages
